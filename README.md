This project performs sentiment analysis on customer reviews using Natural Language Processing (NLP). The reviews are classified as Positive or Negative using Machine Learning techniques.

The project includes:
- Text preprocessing
- Stopword removal
- TF-IDF Vectorization
- Logistic Regression model
- Sentiment prediction and evaluation

Technologies Used:
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- NLTK

The model was trained and evaluated using customer review data to predict sentiments accurately.
